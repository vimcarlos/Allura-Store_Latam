# Allura-Store_Latam
El propósito de este Código es analizar las operaciones (ventas) que 4 tiendas tienen a fin de decidir que tienda será la que deberá cerrar operaciones.
Se realiza un código en colab que inicia cargando los archivos .cvs de que contiene los datos de las ventas de las 4 tiendas, estos datos están organizados por categorías de productos, 
sus precios de venta, el vendedor, cuanto se cobra por enviarlo e incluso la localización geográfica de los clientes.
Para tomar la mejor decisión se analizaron algunos de los datos que se consideraron clave para definir que tienda tiene la menor venta y por ende la menor ganancia.
Se comenzó analizando el total de ventas de cada tienda, ahí se notó que, aunque los montos no son significativamente diferentes una de ellas era la que menos vende. Y aunque las evaluaciones 
de los clientes son muy similares y las mayores ventas suceden en las mismas categorías de productos, los costos de envío evidencian que la tienda 4 ofrece los mas bajos probablemente tratando 
de lograr mejores ventas no lo ha logrado.
Se analizó el comportamiento en el tiempo de las ventas, el comportamiento de los vendedores y finalmente la localización geográfica de los clientes lo que evidencio que es el mismo mercado para 
las 4 tiendas y que no es conveniente mantenerlas todas y es mejor deshacerse de uno o dos. 
Los gráficos de dispersión sirven mucho para observar rápidamente el comportamiento de las ventas (hacia donde se va el producto). 
Incluí varias tablas porque considero que agrupan muy bien la información y permiten una rápida evaluación del comportamiento de las variables analizadas.
